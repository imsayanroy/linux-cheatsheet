## Installation
Here is the process to install Vim in Ubuntu/Kalilinux/Debian etc.
- update the package lists
```bash
sudo apt update
```
- upgrade the package lists
```bash
sudo apt upgrade
```
- install vim
```bash
sudo apt install vim
```
- check if installation is successful
```bash
vim --version
```
vim installation done!
